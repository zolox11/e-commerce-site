# Bestwears

Bestwears is a modern fashion e-commerce website showcasing clothing products online. Products are stored in `products.js` and managed via `main.py`.

---

## Features

- Browse products by categories  
- View product images, prices, and descriptions  
- Dynamic product catalog managed with Python  
- Easy frontend integration  

---

## Product Management

`main.py` allows you to:

- Add new products  
- Update existing products  
- Remove products  

All changes update `products.js`, which the website uses to display products instantly.

---

## Usage

1. Run `python main.py` to manage products  
2. Open the website to browse the updated catalog
